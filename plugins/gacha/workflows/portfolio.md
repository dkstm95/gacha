# gacha portfolio

Use when the user wants to review holdings, concentration, correlation, drawdown, factor exposure, or rebalancing needs.

Workflow:

1. Collect the user's holdings, weights, cost basis, currency, and time horizon if available.
2. Fetch current prices and relevant fund/security metadata.
3. Estimate concentration, overlap, factor/sector/geography exposure, liquidity, volatility, and drawdown risks.
4. Identify overexposure, underexposure, missing hedges, and rebalancing candidates.
5. Provide monitoring triggers and data gaps.

Minimum output:

- holdings summary
- concentration and exposure findings
- risk hotspots
- rebalancing candidates
- missing data
- provenance appendix
