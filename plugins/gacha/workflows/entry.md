# gacha entry

Use when the user has a concrete target and wants to know whether the current price is attractive.

Workflow:

1. Fetch current price, latest financials/fund data, recent news, filings, and key upcoming events.
2. Compare current valuation with historical range, peers, growth expectations, and scenario assumptions.
3. Build bull/base/bear scenarios.
4. Define price zones and tranche logic.
5. State what would invalidate the entry thesis.

Minimum output:

- current price and timestamp
- aggressive buy zone
- first tranche buy zone
- watch/hold zone
- overheated zone
- valuation assumptions
- near-term event risks
- provenance appendix
