# investiq exit

Use when the user owns or tracks an asset and wants sell, trim, stop-loss, or take-profit guidance.

Workflow:

1. Fetch current price, recent news, financial updates, filings, and catalyst changes.
2. Compare current state to the original thesis if provided.
3. Define thesis-based, price-based, and portfolio-based exits.
4. Separate stop-loss, trim, take-profit, and full-exit review zones.
5. Include conditions for continuing to hold.

Minimum output:

- current price and timestamp
- stop-loss or thesis-break zone
- trim zone
- full exit review zone
- conditions to keep holding
- monitoring plan
- provenance appendix
