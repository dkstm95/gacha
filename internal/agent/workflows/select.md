# gacha select

Use when the user has selected a sector, domain, region, theme, or asset class but not a concrete investment vehicle.

Workflow:

1. Define the universe and inclusion/exclusion rules.
2. Search current data for major candidates, ETFs, direct securities, and substitutes.
3. Compare quality, valuation, growth, profitability, leverage, liquidity, fees, concentration, momentum, and risk.
4. Rank candidates and explain why each candidate is included, excluded, or watchlisted.
5. Include a "best fit depends on" section for different user constraints.

Minimum output:

- candidate universe
- comparison table
- ranked recommendation list
- excluded candidates with reasons
- risks and counterarguments
- source log
